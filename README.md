# 🐍 GitHub Contribution Snake

This repository uses GitHub Actions to generate a snake animation from my GitHub contribution graph.

## Light Mode

![GitHub contribution snake](https://raw.githubusercontent.com/Shreya-88/github-snake/output/github-snake.svg)

## Dark Mode

![GitHub contribution snake dark](https://raw.githubusercontent.com/Shreya-88/github-snake/output/github-snake-dark.svg)

## How It Works

- The workflow runs automatically every 12 hours.
- It generates light and dark SVG animations.
- The generated files are published to the `output` branch.
